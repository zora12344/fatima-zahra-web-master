PPE

Pour les sio1 slam, il faut réaliser un client léger (mysql, php) avec :

  • Une authentification (session) ;
    • création d’un compte admin avec tous les droits (CRUD – create, read, update, delete) ;
    • utilisateurs avec seulement des droits de consultations ;
  • réalisation de formulaires qui permettront de visualiser des requêtes sql ;

    Les formulaires peuvent être constitués de :

    • champs texte ;
    • zone de texte ;
    • d’option (radio) ;
    • case à cocher ;
    • liste déroulante à sélection simple ou multiple.

    Les requêtes peuvent être constituées de clause :

    • select d’une ou plusieurs colonnes ;
    • select d’une ou plusieurs colonnes avec une ou plusieurs lignes ;
    • select avec like ;
    • select avec month, year, maintenant() ;
    • select avec order by ;
    • select avec un ou plusieurs agrégats (count, sum, max, min, avg) ;
    • select avec group by ;
    • select avec group by having ;
    • select avec not in ;
    • delete ;
    • update ;
    • insert ;
    • create ;
